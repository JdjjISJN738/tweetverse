# 🚀 TweetVerse AI

AI-powered platform for real-time Twitter sentiment analysis, trending hashtags, and viral content detection.

---

## 🔥 Features

* 📊 Real-time Tweet Sentiment Analysis
* 🔍 Trending Hashtag Detection
* 🤖 NLP-based Classification
* 📈 Dashboard with insights
* ⚡ FastAPI Backend + React Frontend

---

## 🛠 Tech Stack

* Backend: FastAPI
* Frontend: React
* Database: PostgreSQL
* AI: NLP + Machine Learning

## ⚙️ Installation

### 1. Clone Repo

```bash
git clone https://github.com/YOUR_USERNAME/tweetverse-ai.git
cd tweetverse-ai
```

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create `.env` file:

DATABASE_URL=your_db
